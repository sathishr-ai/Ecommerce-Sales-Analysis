Overview:
This project analyzes over 1,200 e-commerce transactions to identify sales trends, customer behavior, and delivery performance.  
Built using Python, Pandas, Seaborn, and Scikit-Learn.

Key Steps:
- Exploratory Data Analysis (EDA)
- KPI Calculation (Revenue, Profit, Late Deliveries)
- Data Visualization
- Random Forest Model for Late Delivery Prediction

Technologies Used:
- Python 3.x  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-Learn  
- Google Colab

Run Instructions:
1. Clone the repo or download files  
2. Open `ecommerce_analysis.ipynb` in Colab  
3. Upload `ecommerce_sales_dataset.csv`  
4. Run all cells
